# Watermarker++

A straightforward image watermarking tool. No bullshit, no cloud services need. Just load an image, add your watermark, export!


## 项目简介

Watermarker++ 是一个简单的图片水印工具，支持文字和图片水印。所有操作均在本地完成，无需上传到云端，保障隐私。

## 功能

**文字水印**
- 自定义文字内容
- 字体大小控制
- 颜色选择器
- 所有参数支持手动输入

**图片水印**
- 本地文件上传或 URL
- 黑白模式切换
- 缩放控制支持手动输入
- 支持常见格式（PNG、JPG、SVG）

**导出**
- PNG 输出
- 保持原图尺寸
- 可调节不透明度


## 开发

```bash
pnpm install
pnpm dev
```

Built with React + TypeScript + Vite.
